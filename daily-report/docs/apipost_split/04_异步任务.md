### 异步任务回调通知

> 创建人: shangyuh

> 更新人: shangyuh

> 创建时间: 2024-02-28 17:08:40

> 更新时间: 2024-02-29 14:10:48

**异步任务的主动回调通知，只有当有回调地址notify_url时候才会通知。,请求方式为POST/JSON。,收到通知后请返回success，否则fail。,通知失败的话最多重试通知5次，每次间隔1分钟。**

**接口状态**

> 已完成

**接口URL**

> 未填写

**请求方式**

> POST

**Content-Type**

> json

**请求Body参数**

```javascript
{
		"task_id": 3,
		"type": "batch_issue_card",
		"msg": "执行任务成功",
		"task_status": 2,
		"org_params": {
			"member_ids": [36],
			"card_list": [{
				"out_ref_no": "CS20240103",
				"card_no": "VID20230831000001",
				"num": 1
			}, {
				"out_ref_no": "YD20240103",
				"card_no": "KID20231023000001",
				"num": 5
			}, {
				"out_ref_no": "CJ20240103",
				"card_no": "VID20231024000001",
				"num": 2
			}]
		},
		"response": [{
			"member_id": 36,
			"msg": "发放异常:会员信息不存在",
			"data": []
		}]
	}
```

| 参数名 | 示例值 | 参数类型 | 是否必填 | 参数描述 |
| --- | --- | ---- | ---- | ---- |
| task_id | 3 | integer | 是 | 任务ID |
| type | batch_issue_card | string | 是 | 业务类型 |
| msg | 执行任务成功 | string | 是 | 任务说明 |
| task_status | 2 | integer | 是 | 异步任务状态 0=新建,1=处理中,2=已处理 |
| org_params | - | object | 是 | 原始请求数据 |
| org_params.member_ids | 36 | array | 是 | - |
| org_params.card_list | - | array | 是 | - |
| org_params.card_list.out_ref_no | CS20240103 | string | 是 | - |
| org_params.card_list.card_no | VID20230831000001 | string | 是 | - |
| org_params.card_list.num | 1 | integer | 是 | - |
| response | - | array | 是 | 回调信息 根据不同的任务返回不同的信息 |
| response.member_id | 36 | integer | 是 | 会员ID |
| response.msg | 发放异常:会员信息不存在 | string | 是 | - |
| response.data | - | array | 是 | - |

**认证方式**

> 继承父级

**响应示例**

* 成功(200)

```javascript
success
```

* 失败(201)

```javascript
fail
```
