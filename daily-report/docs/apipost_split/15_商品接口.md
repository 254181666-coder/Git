> 更新人: wsk

> 创建时间: 2024-04-08 16:14:34

> 更新时间: 2025-10-29 14:25:04

```text
暂无描述
```

**接口状态**

> 已完成

**接口URL**

> /open/product/paging

**请求方式**

> POST

**Content-Type**

> json

**请求Body参数**

```javascript
{
    "shop_id":149,
    "page":1,
    "page_size":100,
    "category_id":0,
    "name":""
}
```

| 参数名 | 示例值 | 参数类型 | 是否必填 | 参数描述 |
| --- | --- | ---- | ---- | ---- |
| shop_id | 149 | integer | 是 | 门店id |
| page | 1 | integer | 否 | 分页页码，数字类型 |
| page_size | 100 | integer | 否 | 每页数量，默认每页100条数据 |
| category_id | 1037 | number | 是 | 分类id |
| name | - | string | 是 | 用户名 |

**认证方式**

> 继承父级

**响应示例**

* 成功(200)

```javascript
{
	"code": 200,
	"sub_code": 0,
	"msg": "",
	"action": "",
	"data": {
		"count": 229,
		"total_page": 3,
		"current_page": 1,
		"page_size": 100,
		"rows": [
			{
				"product_id": 164,
				"shop_id": 149,
				"category_id": 1037,
				"category_name": "酒水",
				"name": "wsk品牌00302",
				"sale_price": 19900,
				"sold_out": 0,
				"vending_machine_produce": 0,
				"disable_system_version": 0,
				"disable_rooms": "{}",
				"policy_version": 0,
				"images": "",
				"status": 1
			}
		]
	}
}
```

| 参数名 | 示例值 | 参数类型 | 参数描述 |
| --- | --- | ---- | ---- |
| code | 200 | integer | - |
| sub_code | 0 | integer | - |
| msg | - | string | 返回文字描述 |
| action | - | string | - |
| data | - | object | 返回数据 |
| data.count | 229 | integer | 总数量 |
| data.total_page | 3 | integer | 分页页码，数字类型 |
| data.current_page | 1 | integer | 分页页码，数字类型 |
| data.page_size | 100 | integer | 每页数量，默认每页100条数据 |
| data.rows | - | array | 商品数据 |
| data.rows.product_id | 164 | integer | 商品id |
| data.rows.shop_id | 149 | integer | 门店id |
| data.rows.category_id | 1037 | integer | 分类id |
| data.rows.category_name | 酒水 | string | - |
| data.rows.name | wsk品牌00302 | string | 商品名称 |
| data.rows.sale_price | 19900 | integer | 价格 |
| data.rows.sold_out | 0 | integer | 估清状态:0=否,1=是 |
| data.rows.vending_machine_produce | 0 | integer | 商品是否支持贩卖机出品:0=否,1=是 |
| data.rows.disable_system_version | 0 | integer | 禁用系统:0=所有系统不禁用,1=收银前台,2=PAD平台,4=手持PDA,8:小程序,16=点歌机 |
| data.rows.disable_rooms | {} | string | 禁用房型,每个门店包厢类型id |
| data.rows.policy_version | 0 | integer | 商品政策:0=所有都允许,1=禁止寄存,2=禁止折扣,4=禁止超售,8=禁止低消 |
| data.rows.images | - | string | 商品图片多张以逗号隔开 |
| data.rows.status | 1 | number | 商品状态:0=禁用,1=启用 |
