**请求Body参数**

```javascript
{
	"shop_id": 67,
	"member_id": 22,
	"card_code": "SUXPSLW3HLSN",
    "remark": "会员资产作废"
}
```

| 参数名 | 示例值 | 参数类型 | 是否必填 | 参数描述 |
| --- | --- | ---- | ---- | ---- |
| shop_id | 67 | string | 是 | 强制核销门店ID |
| member_id | 22 | integer | 是 | 会员ID |
| card_code | SUXPSLW3HLSN | string | 是 | 卡券券码 |
| remark | 会员资产作废 | string | 否 | 备注信息 |

**认证方式**

> 继承父级

**响应示例**

* 成功(200)

```javascript
{
	"code": 200,
	"sub_code": 0,
	"msg": "",
	"action": "",
	"data": {
		"status": true
	}
}
```

| 参数名 | 示例值 | 参数类型 | 参数描述 |
| --- | --- | ---- | ---- |
| code | 200 | integer | - |
| sub_code | 0 | integer | - |
| msg | - | string | - |
| action | - | string | - |
| data | - | object | - |
| data.status | true | boolean | - |

* 失败(404)

```javascript
暂无数据
```
