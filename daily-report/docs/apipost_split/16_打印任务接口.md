**接口状态**

> 已完成

**接口URL**

> /open/print/produce

**请求方式**

> POST

**Content-Type**

> json

**请求Body参数**

```javascript
{
    "shop_id": 67,
    "order_id": 19690
}
```

| 参数名 | 示例值 | 参数类型 | 是否必填 | 参数描述 |
| --- | --- | ---- | ---- | ---- |
| shop_id | 67 | number | 是 | 第三方的门店ID |
| order_id | 19690 | number | 是 | 订单ID |

**认证方式**

> 继承父级

**响应示例**

* 成功(200)

```javascript
{
	"code": 200,
	"sub_code": 0,
	"msg": "",
	"action": "",
	"data": {
		"order_id": 19690,
		"is_send": true
	}
}
```

| 参数名 | 示例值 | 参数类型 | 参数描述 |
| --- | --- | ---- | ---- |
| code | 200 | number | 状态:200=正常,非200=异常 |
| sub_code | 0 | number | - |
| msg | - | string | - |
| action | - | string | - |
| data | - | object | - |
| data.order_id | 19690 | number | 订单ID |
| data.is_send | true | boolean | 是否成功发送消息进行处理 true=是 false=否 |

* 失败(404)
