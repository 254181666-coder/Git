### 获取点单商品列表(分类+商品)

> 创建人: shangyuh

> 更新人: shangyuh

> 创建时间: 2023-11-08 16:59:53

> 更新时间: 2024-04-22 13:55:59

**没有必要传所有查询参数，建议用户用了那些筛选才传对应的参数,当is_optional_pack_type=true的时候，需要商品完全选择好后在调用库存接口查询库存是否满足**

**接口状态**

> 已完成

**接口URL**

> /open/product/list

**请求方式**

> POST

**Content-Type**

> json

**请求Body参数**

```javascript
{
	"shop_id": 67,
	"room_id": 416,
	"member_id": 0,
	"page": 1,
	"page_size": 10,
	"name": "",
	"bar_code": "",
	"category_id": 0,
	"cart_id": "",
	"is_vending_machine": 1
}
```

| 参数名 | 示例值 | 参数类型 | 是否必填 | 参数描述 |
| --- | --- | ---- | ---- | ---- |
| shop_id | 67 | integer | 是 | 门店ID |
| room_id | 416 | integer | 是 | 包厢ID |
| member_id | 0 | integer | 是 | 会员ID 当is_vending_machine=1时可以不传会员ID |
| page | 1 | integer | 否 | 页码 默认为1 |
| page_size | 10 | integer | 否 | 每页条数 默认为10 |
| name | - | string | 否 | 搜索商品名 不搜索则不传 |
| bar_code | - | string | 是 | 搜索国际条形码 完全匹配，不支持模糊搜索 |
| category_id | 0 | integer | 否 | 查询指定分类商品数据 不指定则不传 |
| cart_id | - | string | 否 | 购物车ID 如果有商品数据在购物车需要传入来查询出列表 |
| is_vending_machine | 1 | integer | 否 | 是否为贩卖机点单 0=否 1=是 |

**认证方式**

> 继承父级
