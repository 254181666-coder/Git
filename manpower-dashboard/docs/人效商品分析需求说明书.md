# 人效与商品分析 需求说明书

## 项目概述
对糖果华庭KTV各门店人员效能和商品销售进行数据分析，对比各门店差异，提供优化决策依据。

---

## 一、人效分析需求

### 1. 数据输入
- 人员岗位表：`测试数据/人员岗位.xlsx` - 各门店人员岗位配置
- 储值提成明细表：`测试数据/储值提成明细表_YYYY_MM_DD.xlsx` - 每日储值提成数据
- 商品提成明细表：`测试数据/商品提成明细表_YYYY_MM_DD.xlsx` - 每日商品提成数据
- 日营业数据表：`测试数据/日营业数据表_XXXX.xlsx` - 每日营业数据（营业额、接待台数等）
- 商品销售汇总：`测试数据/商品销售汇总_YYYY_MM_DD.xlsx` - 商品销售汇总数据

### 2. 数据处理规则
- 删除临河街店和总部数据
- 统一门店名称格式：提取简称，佳木斯统一修正为"佳木斯"
- 商品提成和储值提成**分开统计**
- 输出各门店**员工提成明细**

### 3. 分析指标
- 统计各门店总人数、按岗位统计人数
- 计算门店总营业额、当前空包率、总提成（储值+商品分开）
- 核心人效指标：
  - 营业额人效 = 总计营业额 / 总人数
  - 接待台数人效 = 全天待客台数 / 总人数
  - 提成占比 = 当日总提成 / 总计营业额 * 100%
- 提成基尼系数：衡量门店内提成分配差距
  - 0 = 完全平均，1 = 差距极大

### 4. 新增需求
- 增加"提成商品销售总额"列，并计算占商品总销量的比例
- 输出各门店员工提成明细（区分储值/商品）

### 6. 输出要求
- Markdown分析报告
- 独立HTML页面报告，表格清晰展示
- 可视化图表：
  - 各门店营业额人效对比柱状图（通辽店标红）
- 标准化CSV数据存储：
  - `历史数据/YYYY-MM-DD_门店汇总.csv`
  - `历史数据/YYYY-MM-DD_个人提成.csv`
  - `历史数据/YYYY-MM-DD_商品汇总.csv`

## 二、商品分析需求

### 1. 数据输入
- 商品销售汇总：`测试数据/商品销售汇总_YYYY_MM_DD.xlsx`
- 商品销售明细：`商品销售明细_-_商品+包厢维度_YYYY_MM_DD.xlsx`

### 2. 数据处理规则
- 删除临河街数据
- 颗粒度对齐：门店、商品名称关键字一致即对齐

### 3. 分类统计要求

#### (1) 果盘单独统计
- 单独统计**销售**和**赠送**的数量占比、金额占比，特别注意单品售价是0的即为赠送。


#### (2) 重点商品分类销售占比
| 分类 | 包含品类 |
| 酒水 |啤酒、国产啤酒|
| 下酒菜 | 鸭货、小海鲜、冷荤、熟食、拍黄瓜、黄瓜拼盘 |
| 干果 | 干果、坚果、拼盘、优选 |
| 氛围品 | 气氛组、礼品、礼炮、道具、第三方、华子、歌曲 |
| 新品 | 炸食、热食 |

统计各分类销售总额占总销售额的比例及各分类的top10。

### 4. 标杆分析
- 标杆门店：**通辽店**
- 生成通辽店**商品销售结构饼图**（按酒水、下酒菜、干果、氛围品、新品、其他分类）
- 统计各门店每个分类销售占比，对比标杆计算差异（正=高于通辽，负=低于通辽）

### 5. 关联商品分析
- 使用商品销售明细数据，统计同类型包房共现频率，同类型包房的商品结构及关联商品对。


### 6. 输出要求
- Markdown分析报告
- 独立HTML页面报告，表格清晰展示，可视化图表：
  - 通辽店商品销售结构饼图（按酒水、下酒菜、干果、氛围品、新品、其他分类）
- CSV数据：
  - `门店销售结构对比_YYYY-MM-DD.csv`
  - `商品价格对比_YYYY-MM-DD.csv`

- 日期格式：`YYYY-MM-DD`（分析当天日期）

  - `通辽店商品销售结构_YYYY-MM-DD.png` - 商品结构饼图

  - `商品价格对比_YYYY-MM-DD.csv`

- 历史数据目录：`历史数据/`
  - `YYYY-MM-DD_门店汇总.csv`
  - `YYYY-MM-DD_个人提成.csv`
  - `YYYY-MM-DD_商品汇总.csv`
