# 华娱会人效改进完整思路

整合所有会议讨论，梳理出完整的人效改进路径：

---

## 一、顶层逻辑：为什么要改人效？

**核心背景**：

- 市场萎缩，购买力下降，竞争加剧
- 传统KTV人浮于事，人力成本占比越来越高
- 老门店"能人垄断资源"，新人没机会，增长停滞
- 消费者需求升级，需要更好体验，需要更高人效来支撑

**改革目标**：

> 通过合理的任务分配+科学的提成机制，实现：
> 
> 1. 每个人都有事做，没有闲人
> 2. 能者多劳也能多得，激励能干的人
> 3. 新人有成长空间，避免资源垄断
> 4. 总体人力成本下降，人均产出提升，利润增加

---

## 二、调研阶段：摸清现状

### 2.1 需要调研哪些维度

| 维度      | 调研内容                           |
| ------- | ------------------------------ |
| **市场**  | 周边竞品（KTV/酒吧/酒馆/自唱吧）价位、品类、人流    |
| **占比**  | 门店团购/现金/储值卡 三者销售占比             |
| **空包率** | 不同时段（日/黄金/午夜）空包率，找出问题时段        |
| **硬件**  | 音响/麦克/灯光/触摸屏 使用年限、效果，制定更换/维修标准 |
| **商品**  | 酒水/下酒菜/干果/礼炮/蜜饯/膨化 各类别销量/利润占比  |
| **人效**  | 现有工种/编制/工资/工时/福利 逐一梳理          |
| **用工**  | 哪些岗位可以用钟点工/保洁外包，灵活用工           |

### 2.2 调研输出

- 门店现状诊断报告
- 问题点清单：哪些硬件需要换，哪些商品需要调，哪些人岗不匹配
- 改进优先级排序

---

## 三、整改阶段：从硬件到人效，分步落地

### 3.1 第一步：硬件升级（体验打底）

**优先级**：

```
1. 麦克音响 → 2. 灯光 → 3. 触摸屏系统 → 4. 器皿工装 → 5. 陈列布置 → 6. 氛围道具
```

**原则**：

- 能修不换，控制投入成本
- 分区老化分区更换，不一次性全换

### 3.2 第二步：商品结构整改

**核心分级逻辑（SABC）**：

| 分级     | 定位    | 定价策略      | 提成策略 |
| ------ | ----- | --------- | ---- |
| **S级** | 高端酒水  | 溢价合理，保持利润 | 高提成  |
| **A级** | 主流畅销  | 平价走量      | 稳定提成 |
| **B级** | 特色差异化 | 跟随市场      | 中提成  |
| **C级** | 流通小商品 | 低利润引流     | 低提成  |

**品类策略**：

```
高端商品大众化 → 让普通人也消费得起
大众商品品质化 → 提升质感，不比大牌差
品质商品平价化 → 平价也有好品质
```

**场次匹配**：

| 场次      | 客群特点      | 主推商品           |
| ------- | --------- | -------------- |
| **日场**  | 学生/年轻人/团购 | 炸食/水果捞/饮料/特色小食 |
| **黄金场** | 聚会/宴请     | 全品类，主推好酒好菜     |
| **午夜场** | 二场/宵夜     | 简餐/特色小吃/啤酒     |

**核心健康标准**：

```
酒水: 60%-70%
下酒菜+干果: 20%-30%
其他(礼炮/纸抽): 5%-10%
```

## 四、人效整改：定岗定编定额提

### 4.1 分工原则：分工分级，每个人干擅长的

**分级维护**：

| 层级        | 负责客户        | 核心工作       |
| --------- | ----------- | ---------- |
| **店长**    | S级大客户/区域统筹  | 整体统筹，大客户维护 |
| **经理**    | A级客户/团队管理   | 订台办卡，团队管理  |
| **助理/导购** | B/C级新客/商品推销 | 超市商品推销     |
| **迎宾**    | 新进店客人拉新     | 拉新办卡       |

**协作提成模式**（解决内耗，人人都挣钱）：

```
迎宾拉新进店 → 迎宾拿 引荐提成
经理跟进办卡 → 经理拿 主要提成
导购推销商品 → 导购拿 商品提成
每个人做擅长的，每个人都有提成
```

### 4.2 任务分配：基于历史，科学核定

**任务核定公式**：

```
个人任务 = 历史均值 × (1 + 门店增长目标) × 个人能力系数
```

- 能力强、客户多 → 任务相应提高，提成基数也大
- 新人、新岗位 → 任务从低开始，逐步提升

**任务分解**：

- 年度任务 → 分解到月度 → 分解到周日
- 每日跟踪，每周调整，避免月底完不成

### 4.3 提成机制：多劳多得，优质优得

**核心公式**：

```
总提成 =
  (储值任务完成 × 储值提成率) +
  (单提任务完成 × 单提提成率) +
  超额部分 × 超额提成率
```

**分级提成**（引导正确行为）：

| 品类               | 提成比例 | 目的           |
| ---------------- | ---- | ------------ |
| 新办储值卡            | 高    | 鼓励拉新         |
| 转卡/续卡            | 更高   | 鼓励提升客户       |
| 高利润商品（果盘/礼炮/第三方） | 中高   | 鼓励推销高利润      |
| 低利润流通品（纸抽/矿泉水）   | 低    | 不鼓励重点推，但卖了也有 |
| 新品               | 临时提高 | 鼓励推广试错       |

### 4.4 公平性保障：避免垄断，给新人机会

**机会均等算法**：

```python
def opportunity_fairness(employee, category):
    sales_count = get_recent_sales(employee, category)
    max_count = get_max_count(category)
    weight = 1 - (sales_count / max_count)
    # 高价值品类，卖得少的优先获得机会
    if category in ['储值', 'high_value'] and weight > 0.6:
        return True  # 分配机会
    else:
        return False
```

**强制流转机制**：

- 连续 2 个月完不成任务 × 手里客户数超过平均值 → 强制流转部分客户给新人
- 目的：避免"老员工占着坑不产出，新人没机会"

## 五、管理闭环：持续改进

### 5.1 每日

- 系统自动更新：销售/储值/商品/场次 数据
- 店长随时看，每个人都能看到自己进度

### 5.2 每周

- 汇总本周数据
- 对比目标找差距
- 调整任务/人员/提成，下周改进

### 5.3 每月

- 全店汇总分析：
  - 空包率 → 定价/产品对不对？
  - 团购/现金/储值 占比 → 结构健康吗？
  - 每个人任务完成率 → 人效合理吗？
  - 各类商品利润占比 → 商品结构对吗？
- 全员沟通，宣布下月调整
- 审批调整，执行

### 5.4 季度/年度

- 整体定编定员，结构性调整
- 淘汰不合格，补充新鲜血液

## 六、关键诊断指标（人效健康度）

### 个人指标

| 指标        | 计算公式         | 健康标准       |
| --------- | ------------ | ---------- |
| 任务完成率     | 实际完成 / 下达任务  | 平均 70%-80% |
| 储值贡献占比    | 储值提成 / 总提成   | 依门店阶段不同    |
| 商品贡献占比    | 商品提成 / 总提成   | 20%-40%    |
| 新增客户数     | 月新增有效客户      | 持续增长       |
| 续卡率 / 转卡率 | 续卡/到期，转卡/总办卡 | 越高越好       |

### 门店整体指标

| 指标      | 健康标准       | 问题处理              |
| ------- | ---------- | ----------------- |
| 任务完成率分布 | 正态分布       | 两极分化 → 调整任务分配     |
| 提成基尼系数  | 0.3 - 0.4  | > 0.5 → 贫富差距过大，调整 |
| 机会覆盖度   | > 85%      | < 85% → 资源垄断，调整   |
| 客户流转率   | 5%-10% / 月 | 流转率→0 → 僵死，需要强制流转 |

## 七、总结：核心方法论

```
高质低价 → 满足理性消费新趋势
多拉快跑 → 促进流动，不养闲人
数据驱动 → 用数据说话，不靠感觉
持续迭代 → 调研→执行→反馈→调整，滚动改进
```

**一句话总结**：

> 让能者有多得，让新人有机会，让门店有利润，让客户有更好体验

---

## 八、落地步骤建议

1. **第一月**：调研完成，出诊断报告
2. **第二月**：硬件优先整改+商品分级定价+任务定编
3. **第三月**：执行新机制，每周调整
4. **第四月**：总结首月效果，微调后稳定运行

整个改革是**渐进式**，不是一刀切，风险小，见效稳。
