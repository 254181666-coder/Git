# KTV娱乐场所 可视化数据运营体系

## 一、项目概述

### 1.1 服务定位

为KTV、酒吧、密室逃脱、剧本杀等**本地生活娱乐商家**提供一站式数据可视化运营解决方案，通过整合多维度经营数据，构建智能化数据分析看板，帮助商家实现精细化运营管理。

### 1.2 核心价值

- **数据整合**：将分散的Excel、手工台账统一汇聚至数据库
- **实时监控**：多门店经营数据实时可视化展示
- **智能分析**：自动生成日报、周报及专项分析
- **效率提升**：减少人工统计工作量，提升决策效率

## 二、数据看板体系

### 2.1 看板架构

```
┌─────────────────────────────────────────────────────────────┐
│                    Streamlit 可视化看板                        │
├─────────────────────────────────────────────────────────────┤
│  Tab1 📈  │  Tab2 🍺  │  Tab3 👥  │  Tab4 📊  │  Tab5 🔄  │
│ 基础经营  │ 商品销售  │  人效分析  │ 轻舟日报  │  日期对比  │
└─────────────────────────────────────────────────────────────┘
```

### 2.2 模块详细说明

#### 📈 Tab1 - 基础经营数据

| 功能           | 说明                     |
| ------------ | ---------------------- |
| **每日收入趋势**   | 折线图展示每日营业额变化，支持按日期范围筛选 |
| **每日储值金额**   | 展示每日储值卡销售总额及笔数         |
| **商品分类销售趋势** | 多折线图对比酒水、下酒菜、干果等分类销量变化 |
| **通辽店标杆分析**  | 以标杆门店为基准，对比各分类销售结构     |
| **总计统计**     | 饼图+柱状图展示各分类销售额/销量占比    |
| **门店日收入明细**  | 表格展示最新门店收入排名           |

#### 🍺 Tab2 - 商品销售分析

| 功能            | 说明                       |
| ------------- | ------------------------ |
| **分类总览**      | 表格展示各分类商品数、销量、金额、平均单价、占比 |
| **门店分类对比**    | 透视表展示各门店各分类销售额对比         |
| **分类Top10商品** | 可展开查看各分类销量前十商品           |
| **全品类Top10**  | 跨分类统计销量最高的商品             |

#### 👥 Tab3 - 人效分析

| 功能           | 说明                |
| ------------ | ----------------- |
| **门店人效排名**   | 按营业额人效（人均产出）排名    |
| **人效对比图**    | 柱状图可视化各门店人效差异     |
| **标杆门店分析**   | 展示人效最高门店的详细数据     |
| **人效最低门店分析** | 预警提示人效异常门店        |
| **提成基尼系数**   | 分析员工提成分配公平性       |
| **员工提成分析**   | 各门店员工商品提成+储值卡提成一览 |

#### 📊 Tab4 - 轻舟日报分析

| 功能          | 说明                         |
| ----------- | -------------------------- |
| **时段储值率分析** | 分18点前、18-24点、00点后三段分析储值转化率 |
| **储值卡汇总表**  | 各门店储值次数、金额、时段分布明细          |
| **收入结构分析**  | 堆积图+饼图展示储值卡/团购/其他收入占比      |
| **空包率分析**   | 折线图展示各门店包厢闲置率              |

#### 🔄 Tab5 - 日期对比

| 功能          | 说明                     |
| ----------- | ---------------------- |
| **基准期/对比期** | 任意选择两个时间段进行对比          |
| **核心指标对比**  | 总营收、储值金额、待客台数、日均营收环比变化 |
| **详细对比表**   | 表格展示各门店详细数据及变化率        |

## 五、定时任务配置

### 5.1 执行时间

- **每日 10:00** 自动执行（数据通常在营业结束后次日到账）

### 5.2 执行流程

```bash
1. 导入数据库 (import_all_data.py)
   └─> 读取 source/ 目录Excel文件
   └─> 清洗、去重、转换格式
   └─> 写入 MySQL 数据库

2. 生成轻舟日报图表 (generate_charts_qingzhou.py)
   └─> 读取昨日营业数据+储值数据
   └─> 生成储值率分析图 (PNG)
   └─> 生成收入分析综合图 (PNG)
   └─> 保存至 output/ 目录

3. 生成商品销售报告 (generate_product_sales_report.py)
   └─> 读取昨日商品销售数据
   └─> 生成商品销售分析报告 (HTML)
   └─> 保存至 output/ 目录

4. 发送飞书通知
   └─> 推送日报生成状态至企业微信群
```

### 5.3 定时任务命令

```bash
crontab -e
# 添加以下行：
0 10 * * * /path/to/run_daily_routine.sh >> /path/to/logs/cron_routine.log 2>&1
```
