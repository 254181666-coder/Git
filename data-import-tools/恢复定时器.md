## 请手动执行以下命令（按顺序）

打开终端，执行：

```bash
cd /Users/ann/Desktop/AI/Project/数据导入

# 1. 先卸载旧的任务（如果存在）
launchctl unload ~/Library/LaunchAgents/com.ktv.dailyarchive.plist 2>/dev/null || true
launchctl unload ~/Library/LaunchAgents/com.ktv.dailyimport.plist 2>/dev/null || true

# 2. 复制配置文件到 LaunchAgents 目录
cp com.ktv.dailyarchive.plist ~/Library/LaunchAgents/
cp com.ktv.dailyimport.plist ~/Library/LaunchAgents/

# 3. 设置正确的权限
chmod 644 ~/Library/LaunchAgents/com.ktv.dailyarchive.plist
chmod 644 ~/Library/LaunchAgents/com.ktv.dailyimport.plist

# 4. 使用新的 bootstrap 命令加载任务（推荐）
launchctl bootstrap gui/$(id -u) ~/Library/LaunchAgents/com.ktv.dailyarchive.plist
launchctl bootstrap gui/$(id -u) ~/Library/LaunchAgents/com.ktv.dailyimport.plist

# 5. 验证任务是否加载成功
launchctl list | grep com.ktv
```

## 定时任务说明

### com.ktv.dailyarchive (23:00 执行)
- **功能**: 每天23:00自动归档 data/source 文件夹的文件到 data/archive/source_history/
- **保留文件**: 25nian.xlsx 会保留在 source 目录
- **日志**: data/logs/launchd_archive.log 和 launchd_archive.error.log

### com.ktv.dailyimport (09:40 执行)
- **功能**: 每天09:40自动导入数据
- **日志**: data/logs/launchd_import.log 和 launchd_import.error.log
