# Open Questions

Use this file to track questions that should be answered during bring-up.

## Hardware

- How many BU03-Kit boards arrived?
- Do all boards have the same hardware revision?
- What firmware is preloaded?
- What baud rate does the serial interface use?
- Does USB serial work directly on the test computer?
- Does the seller provide a working TWR/PDOA upper-computer tool?
- Can the firmware output distance data without the vendor GUI?

## Positioning

- Does the vendor firmware output only distance, or can it output coordinates?
- How many anchors does the vendor demo require for 2D positioning?
- Does the store require zone-level positioning or shelf-level positioning?
- What is the acceptable error: 10 cm, 30 cm, 50 cm, or 1 meter?
- What update rate is needed?

## Deployment

- Where can anchors be mounted in the store?
- Is wired USB power acceptable for anchors?
- Do tags need batteries?
- How long should a tag last per charge?
- What enclosure size is acceptable?

## Integration

- Which business entities will tags bind to: staff, customers, carts, assets, products, or equipment?
- Does the business system prefer polling API, WebSocket, MQTT, or webhook events?
- Does the store map already exist as an image/CAD file?
- Do we need location history, or only latest position?
