# UWB室内定位与轨迹追踪项目总结

## 项目目标
- 场景：4000平门店（50个包房+大厅+超市）室内人员/物品定位
- 功能：实时定位 + 行动轨迹记录与展示
- 方案：采用成品模块快速搭建

---

## 硬件选型（已确定）

| 参数 | 选型 |
|------|------|
| 模块型号 | NodeMCU-BU01 开发板（安信可） |
| 核心配置 | STM32F103C8T6 + DW1000 UWB芯片 |
| 天线 | 板载PCB天线 |
| 通信方式 | 串口（CP2102 USB-TTL） |
| 固件 | 支持AT指令 |
| 价格 | ¥122.44/块 |

## 硬件连接方案

```
BU01开发板          CP2102 USB-TTL          服务器
    VCC → 3.3V           │
    GND → GND            │
    TX → RX    ──────────┼──→ USB集线器 → 服务器
    RX → TX              │
                         │
                    USB充电头
```

## 基站数量估算
- 总面积约4000平，分隔为50个包房，多遮挡
- 方案：每个包房放置1个基站，大厅/超市按网格布置
- 估算总基站数：**约60-65个**
- 标签数量：根据需要定位的人员/物品数量而定

## 坐标校准方案
- 已有CAD建筑图纸
- 在CAD上选择原点，直接读取每个基站坐标，手动对照安装
- Z坐标统一设置为安装高度（约2.8米）
- 若精度不够，后期可做差分校准

## 注意事项
1. BU01模块使用3.3V供电，请勿接5V
2. 串口连接时TX-RX交叉连接
3. 需要提前在MySQL中创建数据库 `uwb_position`
4. Linux系统可能需要设置串口权限：`sudo chmod 666 /dev/tty.SLAB_USBtoUART*`
5. 每个位置需要被至少3个基站覆盖才能定位
6. 60+设备同时连接，建议使用企业级网络设备
