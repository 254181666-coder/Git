客户续费后，输入服务方提供的短授权码即可恢复。


二、授权中心登录

后台地址：
https://license.xyht618.cn

默认账号：
admin

默认密码：
admin123

首次上线后必须先修改默认密码。

登录后进入“授权管理”页面，可以看到：
1. 生成授权码
2. 账号安全
3. 授权台账


三、修改管理员密码

进入授权中心后，在“账号安全”区域操作：

1. 输入当前密码。
2. 输入新密码。
3. 再次确认新密码。
4. 点击“修改当前账号密码”。

密码要求：
1. 至少 8 位。
2. 同时包含字母和数字。

说明：
当前版本先支持当前登录账号改密码。
管理员不是只能一名，后续可以继续增加“管理员账号管理”，用于新增或停用管理员账号。


四、免费试用与正式授权

0.1.3 安装包默认包含 3 天游试用：

1. 客户首次打开软件时，系统会在客户本机生成试用记录。
2. 试用期从首次启动时间开始计算，持续 3 天。
3. 试用期内可正常使用主要功能。
4. 试用到期后，如果客户没有续费授权，系统会锁定新增、采集、导出等写入功能。
5. 客户续费后，输入短授权码即可恢复使用。

注意：
试用记录保存在客户本机数据目录中，适合一期小规模测试和快速转化。
正式大规模商业化后，建议把试用开通也接入授权中心，由服务端记录设备和试用开始时间。


五、生成客户授权码

进入“生成授权码”区域，填写：

客户名称：
填写客户真实名称，例如“长春某某食品店”。

套餐：
默认会根据授权档位自动填入，例如“30天月度版”。

授权档位：
可选 3 天试用、7 天体验、30 天月卡。

绑定设备码：
可留空。留空时，客户第一次联网激活会自动绑定当前电脑设备码。

备注：
可填写付款状态、渠道来源、销售负责人等，例如“微信收款，张三负责”。

点击“生成短授权码”后，系统会自动复制短码。

日常发给客户使用短码即可。
长码只作为客户网络异常时的离线兜底，不建议日常使用。


六、授权台账操作

授权台账里每一行对应一个客户授权。

主要字段说明：

客户：
客户名称和备注。

短码：
客户日常激活使用的授权码。

套餐/到期：
显示授权档位、到期日期和剩余天数。

状态：
启用表示客户可以激活和校验。
停用表示客户会在联网校验后被限制继续使用写入功能。

激活：
显示激活次数、校验次数、最近激活或校验时间。

操作：
短码：复制短授权码。
续期：选择续 3 天、7 天或 30 天。
停用/启用：控制客户授权状态。
长码：复制长离线码，作为无网络兜底。


七、客户 0.1.3 激活流程

1. 客户安装：
release/木桃工具包-0.1.3-安装包.exe

2. 客户打开软件。

3. 首次打开会自动进入 3 天游试用。

4. 试用期内客户可以先正常测试功能。

5. 客户决定续费后，在软件里点击“重新校验授权”或进入授权页面。

6. 输入服务方提供的短授权码。

7. 点击激活授权。

8. 激活成功后继续使用正式授权。

客户软件联网激活时只上传：
1. 短授权码。
2. 设备码。
3. 软件版本号。
4. 产品名称。

不会上传客户线索、评论、账号登录态、平台账号密码或业务数据。

九、续费流程

客户续费后，在授权中心：

1. 找到客户对应授权记录。
2. 在“续期”下拉中选择续 3 天、7 天或 30 天。
3. 系统会更新到期时间，并自动复制短码。

客户端刷新方式：
1. 客户重新打开软件；或
2. 客户点击“重新校验授权”。

校验成功后，客户本地授权到期时间会更新。


十、停用流程

如果客户试用结束、欠费或需要暂停：

1. 在授权台账找到客户。
2. 点击“停用”。

停用后，客户本地软件不会立刻删除数据。
客户下次联网校验后，会限制新增、采集、导出等写入功能，但仍可查看历史数据。

如果需要恢复：
点击“启用”，客户重新校验授权即可恢复。


十一、服务器后台更新方式

如果只更新授权中心页面或后端逻辑，通常上传覆盖：

/www/wwwroot/mutao-license-center/server.js
/www/wwwroot/mutao-license-center/index.html

然后在服务器执行：

pm2 restart mutao-license-center --update-env

如果只是页面样式更新，只覆盖 index.html 也可以。


十二、验证授权中心是否正常

服务器本机检查：

curl http://127.0.0.1:8795/api/license

公网检查：

curl https://license.xyht618.cn/api/license

正常应看到 status 为 valid，并显示：
授权中心模式已启用，不需要本机客户授权


十三、常见问题

1. 客户安装后为什么不用输入授权码也能进入

0.1.3 安装包已经开启 3 天游试用。
首次打开会自动创建本机试用记录。


2. 试用三天后会怎样

系统会提示试用结束或授权到期。
客户仍可查看历史数据，但不能继续新增、采集、导出或下载备份。
续费后输入短授权码即可恢复。


3. 客户输入短码后提示授权失败

检查：
1. 短码是否复制完整。
2. 授权台账里该短码是否是“启用”状态。
3. 客户电脑是否能访问 https://license.xyht618.cn。
4. 客户是否已经用同一个短码绑定过另一台电脑。


4. 客户换电脑后无法激活

原因：
短码首次激活会绑定设备码。

处理：
当前版本可重新生成一条新的授权码给客户。
后续可以增加“解绑设备”功能。


5. 客户到期后还能打开软件吗

可以打开并查看历史数据。
但不能继续新增任务、采集、导出或下载备份。


6. 长码什么时候用

客户网络异常、无法连接授权中心时才使用长码。
日常交付优先使用短码。


7. 授权中心后台打不开

检查：
1. 服务器 PM2 是否在线：pm2 status
2. Node 服务是否正常：curl http://127.0.0.1:8795/api/license
3. Nginx 是否正常：nginx -t
4. 域名 HTTPS 是否正常：https://license.xyht618.cn


8. HTTPS 证书异常

在宝塔面板中进入：
网站 -> license.xyht618.cn -> SSL

重新申请 Let's Encrypt 证书，并开启强制 HTTPS。


十四、当前测试建议

一期小规模测试建议流程：

1. 先在授权中心修改 admin 默认密码。
2. 先让内部测试人员安装 0.1.3，验证 3 天游试用可正常进入。
3. 用新版 0.1.3 安装包安装客户端。
4. 验证软件登录、账号授权、关键字精准回复、结果复盘。
5. 客户愿意续费后，再在授权中心生成 7 天体验或 30 天月卡短码。
6. 客户输入短码激活正式授权。
7. 在授权中心查看激活时间、设备码和校验记录。
8. 测试续期和停用。

测试通过后，再给真实客户发 7 天体验或 30 天月卡授权。
