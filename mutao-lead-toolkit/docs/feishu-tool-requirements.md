# 飞书工具资料需求评估

## 来源

- 文档：全网AI获客工具「安装包+教程」
- 可见内容：安装包、安装教程、抖音/小红书/快手/B站工具教程、多开、独立 IP、CK、更新、评论监听、企业微信通知、AI 获客建议。

## 可吸收需求

- 工具资料库：记录安装教程、版本、适用系统、常见问题、负责人和验证状态。
- 多平台来源模板：支持抖音、小红书、快手、B站、视频号的评论、直播评论、官方留资、人工导入字段映射。
- 评论监听到内部通知：评论或留资命中规则后，推送给运营或销售处理。
- 安装运维检查：系统要求、安装步骤、更新记录、异常处理和审计日志。
- 运营建议沉淀：把触发词、屏蔽词、地区、来源类型沉淀为发现规则。

## 风险能力处理

以下能力不直接进入产品：

- 多开规避、独立代理 IP、CK 导入。
- 批量关注、批量私信、批量评论、批量 @。
- 绕过平台风控、验证码、设备指纹或登录限制。
- 非授权采集用户个人信息。

对应的合规替代方案：

- 使用官方接口、授权账号、官方留资、人工导入或用户主动互动数据。
- 候选线索必须人工确认后进入线索池。
- 触达动作由员工手动完成，并记录来源、处理人、时间和动作。
- 企业微信只做内部提醒，不代替用户侧自动触达。
