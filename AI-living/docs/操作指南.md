## 二、摄像头使用指南

### 1. 摄像头选择

| 级别 | 推荐型号 | 价格 | 适用场景 |
|------|----------|------|----------|
| 入门 | 罗技 C920 | ¥350 | 个人直播 |
| 进阶 | 罗技 C930e | ¥550 | 专业直播 |
| 专业 | 索尼 A6400 + 采集卡 | ¥4000+ | 企业直播 |

### 2. 摄像头设置

```python
# 摄像头配置示例
config = CameraConfig(
    camera_id=0,           # 摄像头编号（0=默认）
    width=1080,            # 画面宽度
    height=1920,           # 画面高度（竖屏）
    fps=25,                # 帧率
    use_green_screen=False # 是否使用绿幕
)
```

### 3. 绿幕抠图设置

**绿幕参数**：
```python
green_lower_h = 35    # 绿色下限（HSV H）
green_upper_h = 77    # 绿色上限（HSV H）
edge_feather = 3      # 边缘羽化（数值越大越柔和）
spill_suppression = True  # 溢色抑制
```

**调整技巧**：
- 如果抠图不完整 → 调大 `green_upper_h`
- 如果人物被误抠 → 调小 `green_lower_h`
- 边缘有锯齿 → 调大 `edge_feather`
- 人物边缘发绿 → 开启 `spill_suppression`

## 三、平台对接

### 1. 抖音直播

**获取推流地址**：
1. 打开抖音 → 我 → 创作者服务中心
2. 开始直播 → 选择"电脑直播"
3. 复制推流地址（rtmp://...）
4. 替换脚本中的 `RTMP_URL`

**注意事项**：
- 抖音需要实名认证
- 直播前需要完成创作者认证
- 建议先用测试地址验证

### 2. 快手直播

**获取推流地址**：
1. 打开快手 → 我 → 创作者中心
2. 开始直播 → 选择"电脑直播"
3. 复制推流地址

### 3. B站直播

**获取推流地址**：
1. 打开 B站 → 个人中心 → 直播中心
2. 开始直播 → 获取推流码
3. 格式：`rtmp://live-push.bilibili.com/live-bvc/...`

## 四、常见问题

### Q1: 推流断线怎么办？
系统已内置自动重连功能，最多重连 10 次。
可在脚本中调整：
```python
streaming._max_reconnects = 20  # 增加重连次数
streaming._reconnect_delay = 5  # 重连等待时间
```

### Q2: 画面卡顿怎么办？
- 降低分辨率：`width=720, height=1280`
- 降低帧率：`fps=20`
- 降低码率：`bitrate=2000`

### Q3: 绿幕效果不好？
- 确保光线均匀（两侧打光）
- 绿幕要平整无褶皱
- 调整绿幕参数（见第二节）
- 开启溢色抑制

### Q4: LLM 回复太慢？
- 使用更快的模型（如 `qwen-turbo`）
- 减少上下文长度
- 设置合理的 `max_tokens`

## 五、性能优化

### 1. 推流参数建议

| 场景 | 分辨率 | 帧率 | 码率 |
|------|--------|------|------|
| 高质量 | 1080x1920 | 25fps | 3000k |
| 流畅 | 720x1280 | 25fps | 2000k |
| 省流 | 720x1280 | 20fps | 1500k |

### 2. FFmpeg 参数说明

```python
preset = "ultrafast"  # 编码速度（越快CPU占用越低）
# 可选值: ultrafast > superfast > veryfast > faster > fast
```

### 3. 监控指标

系统会自动输出以下指标：
```
=== Stream Stats ===
Frames: 1000              # 总帧数
Dropped: 5 (0.5%)         # 丢帧率
FPS: 24.8                 # 实际帧率
Reconnects: 0             # 重连次数
Uptime: 40s               # 运行时长
====================
```

    "rtmp://快手地址",
