**请求Body参数**

```javascript
{
    "shop_id": 67,
    "cart_id": "product_cart_59272472081d538eddcae6fbc500d0f6",
    "memo": "dddddddddd",
    "member_id": 34,
    "pre_pay": 1,

}
```

| 参数名 | 示例值 | 参数类型 | 是否必填 | 参数描述 |
| --- | --- | ---- | ---- | ---- |
| shop_id | 67 | integer | 是 | 门店id |

| memo | dddddddddd | string | 否 | 开房备注 |
| member_id | 34 | integer | 否 | 会员id |
| pre_pay | 1 | integer | 是 | 1=现结，0=后结 |
| confirm_type | cart | string | 是 | 结算方式，room=开房结算,cart=酒水结算 |
| room_id | 374 | integer | 是 | - |
