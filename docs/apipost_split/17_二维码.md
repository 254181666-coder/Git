### 是否获取企微二维码

> 创建人: wsk

> 更新人: wsk

> 创建时间: 2024-10-24 16:17:20

> 更新时间: 2025-11-28 15:56:26

**没有必要传所有查询参数，建议用户用了那些筛选才传对应的参数**

**接口状态**

> 已完成

**接口URL**

> /open/open_work/is_get_qrcode

**请求方式**

> POST

**Content-Type**

> json

**请求Body参数**

```javascript
{
	"unionid": "o_GTH6R9QVGd8OoPOLubIjbmYOw1",
	"openid": "ovwr04rSkorDrUhC_JN4t5Y54ZYQ",
	"source":"B",
	"shop_id":0,
	"mac":"00:11:22:33:44:01"
}
```

| 参数名 | 示例值 | 参数类型 | 是否必填 | 参数描述 |
| --- | --- | ---- | ---- | ---- |
| unionid | oDsgf6XaZ3-p9JvMnWNcH0QQ0dyU | string | 是 | 小程序unionid |
| openid | o6IVz5XbUJYQgDTVhHTaeqJyVYt0 | string | 是 | 小程序openid |
| source | B | string | 是 | 来源 固定B |
| shop_id | 0 | number | 是 | 门店id |
| mac | 00:71:35:a1:06:11 | string | 是 | 门店id和mac必须传一个 |

**认证方式**

> 继承父级

**响应示例**

* 成功(200)

```javascript
{
	"code": 200,
	"sub_code": 0,
	"msg": "",
	"action": "",
	"data": {
		"res": {
			"is_follow": 1,
			"url": "https://wework.qpic.cn/wwpic3az/530920_0q2Uz6SqRWurWAM_1723791352/0"
		}
	}
}
```

| 参数名 | 示例值 | 参数类型 | 参数描述 |
| --- | --- | ---- | ---- |
| code | 200 | number | - |
| sub_code | 0 | number | - |
| msg | - | string | 返回文字描述 |
| action | - | string | - |
| data | - | object | 返回数据 |
| data.res | - | object | - |
| data.res.is_follow | 1 | number | 0不需要关注企微 1需要 |
| data.res.url | https://wework.qpic.cn/wwpic3az/530920_0q2Uz6SqRWurWAM_1723791352/0 | string | is_follow为1返回二维码 |
